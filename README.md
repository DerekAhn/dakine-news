# dakine-news

Simple App built on Go's [Gin framework](https://gin-gonic.github.io/gin/), that deliver's Oahu's latest surf report.


### Run App
```bash
$ go build
$ ./dakine-news

```
### PWA Experience
- Either run application or visit [dakine](https://dakine.news).
